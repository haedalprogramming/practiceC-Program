## 4. 예시: 간단한 메모장 프로그램

### 기획
- **목적**: 간단한 텍스트 메모를 저장하고 불러오기
- **기능**: 
  - 메모 작성
  - 메모 목록 보기
  - 메모 삭제
  - 프로그램 종료

### 설계
```c
// 필요한 변수
char memos[100][200];  // 메모 저장 배열
int memo_count = 0;    // 현재 메모 개수

// 필요한 함수
void show_menu();      // 메뉴 출력
void add_memo();       // 메모 추가
void show_memos();     // 메모 목록 출력
void delete_memo();    // 메모 삭제
```

### 구현 예시
<details>
<summary>코드 보기</summary>

```c
#include <stdio.h>
#include <string.h>

char memos[100][200];
int memo_count = 0;

void show_menu() {
    printf("\n============================\n");
    printf("       [간단 메모장]        \n");
    printf("============================\n");
    printf("1. 메모 작성\n");
    printf("2. 메모 목록\n");
    printf("3. 메모 삭제\n");
    printf("4. 종료\n");
    printf("============================\n");
    printf("선택: ");
}

void add_memo() {
    if (memo_count >= 100) {
        printf("메모장이 가득 찼습니다!\n");
        return;
    }
    
    printf("메모를 입력하세요: ");
    getchar(); // 버퍼 비우기
    fgets(memos[memo_count], 200, stdin);
    
    // 개행문자 제거
    memos[memo_count][strcspn(memos[memo_count], "\n")] = 0;
    
    memo_count++;
    printf("메모가 저장되었습니다!\n");
}

void show_memos() {
    if (memo_count == 0) {
        printf("저장된 메모가 없습니다.\n");
        return;
    }
    
    printf("\n--- 메모 목록 ---\n");
    for (int i = 0; i < memo_count; i++) {
        printf("%d. %s\n", i+1, memos[i]);
    }
}

void delete_memo() {
    if (memo_count == 0) {
        printf("삭제할 메모가 없습니다.\n");
        return;
    }
    
    show_memos();
    int num;
    printf("삭제할 메모 번호: ");
    scanf("%d", &num);
    
    if (num < 1 || num > memo_count) {
        printf("잘못된 번호입니다.\n");
        return;
    }
    
    // 메모 삭제 (뒤의 메모들을 앞으로 이동)
    for (int i = num-1; i < memo_count-1; i++) {
        strcpy(memos[i], memos[i+1]);
    }
    memo_count--;
    printf("메모가 삭제되었습니다.\n");
}

int main() {
    int choice;
    
    while (1) {
        show_menu();
        scanf("%d", &choice);
        
        switch (choice) {
            case 1: add_memo(); break;
            case 2: show_memos(); break;
            case 3: delete_memo(); break;
            case 4: 
                printf("프로그램을 종료합니다.\n");
                return 0;
            default:
                printf("잘못된 선택입니다.\n");
        }
    }
}
```

## 5. 프로그래밍 팁 💡

### 🚀 개발 효율성 높이기
1. **작은 단위로 나누어 개발**
   - 큰 프로그램도 작은 기능들의 조합
   - 하나씩 완성하고 다음 기능으로

2. **자주 테스트하기**
   - 새 기능 추가할 때마다 테스트
   - 오류는 빨리 발견할수록 쉽게 해결

3. **주석 활용하기**
   - 복잡한 부분에는 설명 주석
   - 나중에 자신이 봐도 이해할 수 있게

### 🐛 오류 해결하기
1. **컴파일 오류**
   - 문법 오류, 변수명 오타 확인
   - 세미콜론, 중괄호 확인

2. **논리 오류**
   - printf로 중간값 출력해보기
   - 조건문, 반복문 조건 다시 확인

3. **실행 오류**
   - 배열 범위 초과 주의
   - 0으로 나누기 주의

### 📚 추가 학습 방향
1. **파일 입출력** - 데이터를 파일에 저장
2. **구조체** - 복잡한 데이터 관리
3. **포인터** - 더 효율적인 메모리 사용
4. **동적 메모리** - 가변 크기 데이터 처리
